TYPECHO响应式主题 :Autumn 1.0 发布

发表于 2016-09-28   |   分类于 Typecho

这是一款仿自然智Typecho模板，历时2两个星期，终于发布了，模板还有很多地方需要完善，后期我也会一直更新，直到没有人用这套主题为止。

详情

预览地址：http://www.3gjn.com
最新下载地址：Autumn.zip

关于

处女作主题命名为Autumn（秋天）

使用方法

下载主题拷贝到 themes 模板目录，进入后台启用模板即可。
